# The Essential Cure 🌿

A premium, high-aesthetic e-commerce platform for luxury hair care.

## 🚀 Deployment Essentials
1. **Database**: Apply all `.sql` files in the Supabase SQL editor (idempotent).
2. **Environment**:
   - `VITE_SUPABASE_URL`: Your Supabase URL.
   - `VITE_SUPABASE_ANON_KEY`: Your Supabase Public Key.
3. **Build**: `npm run build`

## 🛠 Features
- **Luxury Frontend**: Custom-themed UI for a premium experience.
- **Hair Quiz**: Dynamic lead collection and product recommendations.
- **Transformation Hub**: SEO-optimized "Before & After" gallery.
- **Admin Suite**: Advanced Order, Product, Discount, and Lead management.
- **Loyalty System**: Referral and purchase-based points system.

## 📈 Store Operations
- **Analytics**: Real-time sales charts and regional dominance reports.
- **Order Tracking**: Customer-facing real-time tracking with security verification.
- **Image Hub**: Automatic compression for all admin-uploaded media.

Built with React, Vite, Tailwind CSS, Framer Motion, and Supabase.
